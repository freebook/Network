防止日志文件本修改，加入某些属性。

1：chattr 命令可以使某个文件属性改变
     chattr  +a /var/log/array.log   只允许追加，不允许减少。
     chattr  -a /var/log/array.log   去掉属性。
2：lsattr 命令查看chattr属性
# lsattr /var/log/array.log
-----a------- /var/log/array.log
